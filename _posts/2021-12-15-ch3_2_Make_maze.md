### 1. 가장자리 벽으로 채우기

```csharp
public void Initialize(int size)
{
    _tile = new TileType[size, size];
    _size = size;

    for (int y = 0; y < _size; y++)
    {
        for (int x = 0; x < _size; x++)
        {
            if (x == 0 || x == _size - 1 || y == 0 || y == size - 1)
                _tile[y, x] = TileType.Wall;
            else
                _tile[y, x] = TileType.Empty;
        }
    }
}
```
